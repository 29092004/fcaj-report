Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - AI GATEWAY: GIẢI PHÁP QUẢN LÝ AI TRÊN AWS VỚI AMAZON BEDROCK VÀ AMAZON API GATEWAY](3.1-Blog1/)
Mô hình AI Gateway sử dụng Amazon API Gateway làm lớp truy cập phía trước Amazon Bedrock. Giải pháp hỗ trợ nhiều tính năng quan trọng như xác thực, phân quyền người dùng, quản lý hạn mức sử dụng (throttling), quản lý vòng đời API, và tích hợp AWS WAF nhằm kiểm soát chi phí và tăng cường bảo mật cho Generative AI.

###  [Blog 2 - TỐI ƯU HÓA VẬN HÀNH ĐÁM MÂY VỚI GIẢI PHÁP KIRO CHO AWS DEVOPS AGENT](3.2-Blog2/)
Giải pháp kết hợp môi trường phát triển tích hợp (IDE) Kiro hỗ trợ bởi AI của Amazon với AWS DevOps Agent. Giải pháp mang toàn bộ dữ liệu vận hành thực tế từ AWS vào giao diện IDE quen thuộc, giúp tự động hóa quá trình phát hiện, phân tích và khắc phục sự cố một cách nhanh chóng.

###  [Blog 3 - AMAZON S3 ANNOTATIONS: GẮN NGỮ CẢNH PHONG PHÚ VÀ CÓ THỂ TRUY VẤN TRỰC TIẾP LÊN OBJECT](3.3-Blog3/)
Giới thiệu tính năng Amazon S3 Annotations cho phép đính kèm thông tin nghiệp vụ và ngữ cảnh phong phú trực tiếp vào từng S3 object lên tới 1 GB. Các chú thích này có cấu trúc độc lập, dễ dàng thay đổi mà không cần ghi đè file gốc và hỗ trợ truy vấn trực tiếp bằng SQL thông qua Amazon Athena.
