During my internship at **Amazon Web Services Vietnam Co., Ltd.** from **April 17, 2026** to **July 12, 2026**, I had the opportunity to learn, practice, and apply the knowledge acquired in school to a real-world working environment.

Specifically, my participation and contributions during this period include:
* **Studying and researching AWS Cloud Services:** Gaining hands-on familiarity with core cloud infrastructure concepts including Virtual Private Clouds (**VPC**), Identity and Access Management (**IAM**), **Serverless** application design, Infrastructure-as-Code deployments via **AWS SAM**, and client-side hosting and delivery through **AWS Amplify**.
* **Developing the Jobs-Matching-Platform Project:**
  * Designing and building responsive frontend user interfaces using the modern Next.js framework to deliver a clean, intuitive, and seamless user experience.
  * Integrating API routing to fetch and bind dynamic datasets from the **DynamoDB** database to populate the **Home Page** and the **Job Detail Page**.
  * Researching retrieval mechanics and directly implementing the fast **Job Search and Filtering** feature to maximize user engagement and system utility.
  * Designing infrastructure and deploying the Backend using **AWS SAM**: writing the infrastructure configuration in the **template** file (`template.yaml`) to define AWS cloud resources (API Gateway, Lambda, DynamoDB, Cognito User Pool, S3 Bucket) and executing deployments to the Cloud.
  * Executing **Frontend Hosting** using **AWS Amplify Hosting**, configuring Git repository integrations, setting up automated CI/CD pipelines, and setting up client **environment variables** (`NEXT_PUBLIC_API_BASE_URL`, etc.) within the Amplify Console to securely connect to Backend APIs.
  * Setting up and managing user authentication workflows using **Amazon Cognito**: integrating Sign-up/Sign-in flows (Cognito Managed Login) and token management to protect client-side routes (Routing Protection) in Next.js such as `/favorites` and `/upload-cv`.
  * Configuring secure API endpoint authorization by setting up a **Cognito JWT Authorizer** on **API Gateway**, alongside managing Cross-Origin Resource Sharing (**CORS**) policies on **Amazon S3** to allow secure candidate CV uploads.

Through hands-on practical activities during this internship, I have significantly improved and solidified the following core skillsets:

* **Technical Skills:**
  * **Frontend Development & UI Engineering:** Mastering the **Next.js** framework and **JavaScript** to build performant web pages (Home, Job Details), optimizing DOM structure, and understanding rendering strategies like Server-Side Rendering (SSR) and Client-Side Rendering (CSR).
  * **System Data Integration:** Gaining proficiency in integrating API endpoints to request and fetch dynamic data from **Amazon DynamoDB** databases, parsing complex JSON responses, and binding them accurately to frontend components.
  * **Serverless Architecture & Infrastructure-as-Code:** Learning how to write and organize AWS SAM infrastructure files (`template.yaml`) to declare resources (Lambdas, API Gateways, DynamoDB tables), package applications, and successfully deploy them to the AWS Cloud.
  * **Application Hosting & CI/CD Pipelines:** Mastering automated CI/CD configurations through **AWS Amplify Hosting**, connecting Git branches for auto-triggered builds, and managing secure client **Environment Variables** within the Amplify Console.

* **Soft Skills & Professional Qualities:**
  * **Technical Problem-Solving Mindset:** Enhancing debugging capabilities by analyzing logs (CloudWatch, browser console) during API integration, investigating errors, and optimizing job search and filter algorithms.
  * **Independent Technology Research:** Developing the ability to read and comprehend official AWS English documentation, whitepapers, and developer guides to independently resolve complex technical challenges.
  * **Bilingual Technical Writing:** Learning to write and maintain clear, well-structured technical user guides using Markdown format, ensuring strict alignment and synchronization between English and Vietnamese versions.
  * **Agile Teamwork & Collaboration:** Adapting to Agile/Scrum workflows, using project tracking tools, proactively reporting progress, and communicating system blockers with mentors to co-design technical solutions.
